# ios-news-app

# Overview

    The ios-news-app is a app for view news in several countries. It was built to test some knowledge in iOS.

# How it works

    To test this app
    - Create a key at https://newsapi.org
    - In class `NewsService` replace value `parameters["apiKey"] = "your key"`
    - Run this app
